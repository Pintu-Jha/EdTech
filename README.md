# EdTech Assignment

## Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/Pintu-Jha/EdTech.git
   cd EdTechAssignment
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables** (If required)
   ```bash
   cp .env.example .env
   ```

4. **Start the development server**
   ```bash
   npm run start
   ```

5. **Run the app**
   Once the server starts, press one of the following in your terminal:
   - `i` for iOS Simulator
   - `a` for Android Emulator
   - Or scan the QR code using the Expo Go app on your phone
